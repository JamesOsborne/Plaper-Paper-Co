## Branch Guidelines

| Type     | Name | From    | To      |
| -------- |:----:| ------- | ------- |
| Feature  | ft-* | develop | develop |
| Hotfix   | hf-* | master  | any     |
| Release  | rl-* | develop | master  |

Create new branch for everything. Push it and submit a pull request to get it merged.
